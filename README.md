# University Payroll System

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)
![MySQL](https://img.shields.io/badge/MySQL-000000?style=for-the-badge&logo=mysql&logoColor=white)

**A full stack Payroll Management System designed for academic institutions.**
This application automates personnel management and complex compensation logic, ensuring accurate monthly reporting and data integrity.

### ‚ö° Key Features

* **Employee Management:** Handles distinct roles (**Administrative vs. Academic**) and employment types (**Permanent vs. Contract**) with strict validation logic.
* **Automated Payroll Engine:** Calculates salaries based on tenure (**15% increase/year**), family status, and specific allowances (Research/Library).
* **Financial Reporting:** Generates monthly payment statements and analytics for min/max/average salaries per category.
* **Data Integrity:** Built on a **3NF normalized MySQL database**, utilizing Stored Procedures and Views for security and speed.
